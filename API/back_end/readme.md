
# API - SEQUELIZE
# BACKEND
Esse arquivo é o projeto para API que fornece serviço de controle do bando de dados para registro de usuários e suas relações.

## COMO EXECUTAR

- Edite o arquivo .env.example com as suas informações do banco (nome,
senha e usuário), depois renomei o arquivo para apenas .env

- Uma vez que o repositório esteja em sua máquina local e o servidor do banco SQL esteja rodando, abra o terminal dentro da pasta com esses arquivos e execute o comando:

```
npm install
npm run start:dev
```

Se tudo ocorreu como o esperado a mensagem 'App listening on port 8000'
